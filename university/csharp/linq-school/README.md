# LINQ School

C# console application with LINQ queries on school data — students, classes, exams, and grades.

## Run

Open `LinqSchool.csproj` in Visual Studio.

```bash
# from this folder, if dotnet supports the project
dotnet build LinqSchool.csproj
```

.NET Framework 4.7.2 — Visual Studio on Windows recommended.

## Files

| File | Role |
|------|------|
| `Program.cs` | LINQ queries |
| `Escola.cs` | Data model |
| `Utils.cs` | Helpers |
