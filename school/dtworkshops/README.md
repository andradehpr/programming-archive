# dtworkshops

Console application for a car workshop — appointments, customer and vehicle data, bilingual menus.

**When:** 1st year of secondary school (2021)

## Run

Open `src/finalproject.cbp` in Code::Blocks, or:

```bash
cd src
gcc -o dtworkshops main.c
./dtworkshops
```

## Structure

| Path | Description |
|------|-------------|
| `src/main.c` | Application logic |
| `src/func.h` | Helpers and ASCII art |
| `src/dados.txt` | Sample data |
| `docs/` | Manuals |

Requires `conio.h` on Windows (see `docs/Manual_Utilizador/Conio/`).
