# WPF Toolkit

WPF desktop application — BMI calculator, dice game, converter, and main menu.

## Run

Open `WPF1.csproj` in Visual Studio (.NET Framework 4.8.1).

## Structure

```
Controller/IMC_VM.cs   # ViewModel
Controller/Cmd.cs      # Commands
IMC.xaml
JogoDados.xaml
MainWindow.xaml
```
