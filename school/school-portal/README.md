# school-portal

WinForms desktop application with SQL Server — login, faculty, alumni, and bibliography modules.

**When:** 2nd year of secondary school (2022)

## Run

Open `src/ProjetoF/ProjetoF.csproj` in Visual Studio (Windows).

Configure the connection string in `src/ProjetoF/App.config` for your local SQL Server.

## Structure

```
src/ProjetoF/
├── Form1.cs          # Login
├── adminform.cs      # Admin
├── FACULS.cs         # Faculty
├── ALUMNI.cs         # Alumni
└── bibliografia.cs   # Bibliography
```
