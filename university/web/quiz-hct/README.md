# Quiz HCT

Multiple-choice quiz with 80 questions on History of Science and Technology.

**When:** Built on 26 June 2026 as a personal study tool for the HCT course — not a graded assignment.

## Run

```bash
open index.html
# or
python3 -m http.server 8080
```

## Files

| File | Role |
|------|------|
| `index.html` | Structure |
| `style.css` | Styles |
| `script.js` | Logic and question data |

Add or edit questions in the `QUESTIONS` array in `script.js`.
